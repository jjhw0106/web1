*진행 순서(데이터 조회)
1. 로그인 : -uroot -p
2. 어떤 데이터베이스(스키마) 쓸거니? : USE 데이터베이스 명
3. SHOW DATABASES : 모든 데이터베이스를 보여줌
4. SHOW TABLES : 만들어져 있는 테이블 보여줌
5. SELECT * FROM TABLE명 : 테이블에서 모든 정보 가져옴

*명령어 (대문자가 명령어, 소문자는 내가 지정한 이름들)
SHOW DATABASES: 데이터 베이스를 보여달라
USE opentutorials(스키마 이름) : opentutorials스키마에서 작업을 하겠다.
NULL/NOT NULL : 빈 값 허용한다 / 빈 값을 허용하지 않겠다.
AUTO_INCREMENT : 값을 1씩 올림(키값 등에 사용)
DESC 테이블명 : 테이블의 구조를 보여줌
RENAME TABLE topic TO topic_backup : 테이블 이름 변경
SELECT topic.id AS topic_id~: topic테이블의 id를 topic_id로 바꿔서 출력해라.

테이블 삭제
DROP TABLE 테이블명

테이블 만들기
*CREATE TABLE topic(
id INT(11) NOT NULL AUTO_INCREMENT(첫번째 열 id, 인트형 11사이즈, 널허용x, 숫자 자동증가),
title VARCHAR(100) NOT NULL,
created DATETIME NOT NULL,
...
);

*INSERT (열 추가)

*JOIN
SELECT * FROM LEFT JOIN author ON topic.author_id = author.id;
: author테이블과 topic에서 id가 같은 것들을 합쳐서 조회해라
